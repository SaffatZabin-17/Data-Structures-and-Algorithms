*********ReadMe for BST Offline************


1. The BST ADT contains 4 operations. Insert(I), Delete(D), Find(F), and Traversal (T)

2. All the operations use recursive calls to complete their operations.

3. There are codes for 3 types of Traversal. Pre-order, In-order and Post-order traversal

4. The code reads input from a file name "Input_BST.txt". 

5. Always make sure the input file is in the directory outside src
