User Instrcutions:

1. The zip file contains source code to implement stack in two ways. First method includes an array based

   implementation, and the second method includes linked list based implementation.

2. Appropriate main function has been included to demonstrate the operations listed in the stack ADT

3. To demonstrate the stack ADT implementation in real life scenario, a dishwasher simulation has been

   created. The dishwasher simulation has been done in three ways. First one includes array based implementation

   demonstration, second one includes linked list based implementation demo, and third one includes one

   array two stack method of dishwasher simulation.
